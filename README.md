# task-cli
Проста To-Do утиліта з CLI 

To-Do утиліта з використанням sqlite. 

# Як встановити 
1) Завантажте утиліту через `github` або через `git clone`:
```bash
git clone https://github.com/dimas-52/task-cli.git
```
2) Перейдіть в директорію утиліти:
```bash
cd task-cli
```
3) Зробіть файл виконуваним:
```bash
chmod +x main.py
```
4) Створіть символічне посилання в `/usr/local/bin` для того щоб утиліту можна було викликати з терміналу:
```bash
sudo ln -s "$(pwd)/main.py" /usr/local/bin/tas
```

# Як використовувати?
1) Спочатку потрібно створити коритувача, введіть назву користувача, нарпиклад своє `імя` або своє `імя_робота` використавши команду:
```bash
tas --create_user
```
2) Виберіть стандартного користувача командою:
```bash
tas --default_user
``` 
- тепер ви ви можете створити нотатку, командою:
```bash
tas -a "інформація для нотатки"
```
- Перегляньте нотатки за допомогою команди:
```bash
tas -v
``` 

- `-a`, `--add "текст нотатки"` - додати нотатку
- `--create_user` - створити користувача
- `--default_user` - вибір стандартного користувача
- `-v`, `--view` - перегляд нотатків

# Як працює ця утиліта? 
На початку роботи утиліти вона створює реляційну базу данних за допомогою SQLite і в БД створює 4 таблиці(users, categories, main, settings). 

**1. Таблиця `users`** (Профілі користувачів):
| id | username | created_at (Unix Timestamp) |
| :--- | :--- | :--- |
| 1 | user | 1769974860 |
| 2 | user_work | 1769974893 |

**2. Таблиця `categories`** (Довідник типів записів):
| id | name |
| :--- | :--- |
| 1 | idea |

**3. Таблиця `main`** (Центральна таблиця зв'язків):
| id | user_id(посилання на users(таблиця 1) | category_id(посилання на categories(таблиця 2)) | content |  is_done(1 якщо виконано або видалено) | created_at (Unix Timestamp) |
| :--- | :--- | :--- | :--- | :--- |  :--- |
| 1  | 1 | 1 | потрібно дізнатись як працює БД | 0 | 1769975614 | 
| 2  | 2 | 1 | потрібно попросити підвищення | 1 | 1769975614 | 

**4. Таблиця `settings`** (Поточний стан програми):
settings:
| id | current_user_id(посилання на таблицю users(таблиця 1)) |
| :--- | :--- |
| 1 | 1 |
